# Programming Exercise: Vending Machine Exercise

Design a vending machine using a programming language of your choice. 

### The vending machine should perform as follows:
* Once an item is selected and the appropriate amount of money is inserted, the vending machine should return the correct product.
* It should also return change if too much money is provided, or ask for more money if insufficient funds have been inserted.
* The machine should take an initial load of products and change. 
* The change will be of denominations 1p, 2p, 5p, 10p, 20p, 50p, £1, £2. There should be a way of reloading either products or change at a later point.
* The machine should keep the state of products and change that it contains up to date.


Your project will be part of a more complex system, and it will be handed over to a team for maintenance and the development of new features. Thus, structuring the project correctly in terms of code, tests, and documentation, is part of the exercise.


Note: You do not need to create any visual UI for this task.



Please develop the machine in any language and platform (such as GitLab GitHub) you are comfortable with.
